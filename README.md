# NPCBotPanel
NPCBot Panel es un addon ligero para World of Warcraft en servidores privados que usan el mod NPCBots. En lugar de escribir comandos en el chat cada vez, tendrás un panel visual con botones organizados en pestañas.

# 🇪🇸 Español
¿Qué es NPCBot Panel?
NPCBot Panel es un addon ligero para World of Warcraft en servidores privados que usan el mod NPCBots. En lugar de escribir .npcbot command standstill en el chat cada vez, tendrás un panel visual con botones organizados en pestañas.
✨ Características

🖱️ Botones clickables para todos los comandos principales de NPCBot
📂 4 pestañas organizadas — Movimiento, Combate, Vínculos, Utilidad
🌐 Interfaz bilingüe — cambia entre Español e Inglés en el juego con un desplegable
📌 Botón en el minimapa — arrástralo a donde quieras alrededor del minimapa
🔒 Bloquear/Desbloquear — fija el panel para que no se mueva accidentalmente
💾 Posición guardada — el panel recuerda dónde lo dejaste entre sesiones
🛠️ Tooltips — pasa el ratón sobre cualquier botón para ver su descripción y el comando exacto que ejecuta
👤 Creado por Lleguito

# 📋 Requisitos
Cliente WoW3.3.5a (Wrath of the Lich King)
Core del servidorAzerothCore o TrinityCore
Mod NPCBotstrickerer/Trinity-Bots
PermisosLos comandos requieren los permisos apropiados en tu servidor

# 📦 Instalación

Descarga la última versión y extráela.
Copia la carpeta NPCBotPanel dentro de tu directorio de addons:

   World of Warcraft/Interface/AddOns/NPCBotPanel/

Asegúrate de que la carpeta contiene estos dos archivos:

   NPCBotPanel/
   ├── NPCBotPanel.toc
   └── NPCBotPanel.lua

Lanza el juego y haz clic en AddOns en la pantalla de selección de personaje.
Asegúrate de que NPCBot Panel está activado y haz clic en Entrar al mundo.

# 🇬🇧 English
What is NPCBot Panel?

NPCBot Panel is a lightweight World of Warcraft addon for private servers running the NPCBots mod. Instead of typing commands in the chat every time, you get a clean in-game panel with clickable buttons organised into tabs.

# ✨ Features

🖱️ Clickable buttons for all major NPCBot commands
📂 4 organised tabs — Movement, Combat, Bindings, Utility
🌐 Bilingual UI — switch between English and Spanish in-game via a dropdown
📌 Minimap button — drag it anywhere around the minimap
🔒 Lock/Unlock — pin the panel so it doesn't move accidentally
💾 Persistent position — panel remembers where you left it between sessions
🛠️ Tooltips — hover any button to see its description and the exact command it runs
👤 Created by Lleguito

# 📋 Requirements
Client3.3.5a (Wrath of the Lich King)
Server coreAzerothCore or TrinityCore
NPCBots modtrickerer/Trinity-Bots
GM levelCommands require appropriate GM/player permissions on your server

# 📦 Installation

Download the latest release and extract it.
Copy the NPCBotPanel folder into your addons directory:

   World of Warcraft/Interface/AddOns/NPCBotPanel/

Make sure the folder contains these two files:

   NPCBotPanel/
   ├── NPCBotPanel.toc
   └── NPCBotPanel.lua

Launch the game and click AddOns on the character selection screen.
Make sure NPCBot Panel is enabled and click Enter World.
